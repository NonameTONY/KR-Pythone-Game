# 2D Platformer на Pygame

## Требования

- Python 3.10+
- Библиотека Pygame: pip install pygame

## Запуск игры

1. Убедитесь, что файл карты `level01.json` лежит в одной папке с `main.py`.
2. Запустите игру командой:
   python main.py

## Управление в игре

- A или стрелка влево  — движение влево
- D или стрелка вправо — движение вправо
- W, стрелка вверх или пробел — прыжок
- Left Shift или X     — рывок вперёд
- Esc                  — выход из игры

Цель уровня — добежать до клетки-финиша, отмеченной особыми блоками.

## Запуск редактора уровней

1. Запустите редактор командой:
   python editor.py
2. После редактирования сохраните карту в `level01.json` с помощью горячей клавиши.

## Управление в редакторе

- Левая кнопка мыши  — поставить блок
- Правая кнопка мыши — удалить блок
- Колёсико мыши      — переключение типа блока
- W / A / S / D      — перемещение камеры по уровню
- S                  — сохранение карты в JSON (level01.json)
- L                  — загрузка карты из JSON (level01.json)
- G                  — показать / скрыть сетку
- Esc                — выход из редактора
