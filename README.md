# 2D Платформер на Pygame

Учебный 2D-платформер с тайловой картой, прыжками, рывком персонажа и встроенным редактором уровней, реализованный на Python с использованием библиотеки Pygame.

---

## Требования к окружению

- Python 3.10 и выше
- Библиотека Pygame

Установка зависимостей:

```bash
pip install pygame
```

---

## Запуск игры

1. Убедиться, что файл уровня `map.json` находится в той же директории, что и `main.py`.
2. Запустить игру командой:

```bash
python main.py
```

---

## Управление в игре

| Клавиша | Действие |
|---|---|
| `←` / `A` | Движение влево |
| `→` / `D` | Движение вправо |
| `↑` / `W` / `Space` | Прыжок |
| `X` | Рывок в направлении движения |
| `Esc` | Выход из игры |

Цель игрока — добраться до финишного блока уровня. При успешном прохождении отображается сообщение **«Level passed!»**.

---

## Запуск редактора уровней

Редактор уровней запускается отдельной программой:

```bash
python editor.py
```

---

## Управление в редакторе

| Клавиша / Кнопка | Действие |
|---|---|
| `W` `A` `S` `D` | Перемещение камеры по уровню |
| `ЛКМ` | Установить тайл текущего типа |
| `ПКМ` | Удалить тайл в выбранной клетке |
| Колесо мыши | Переключение группы тайлов |
| `Shift` + колесо | Переключение варианта тайла |
| `G` | Включить / выключить сетку (on-grid / off-grid) |
| `O` | Сохранить карту в файл `map.json` |
| `T` | Автозаполнение тайлов (autotile) |
| `Esc` | Выход из редактора |

## Структура проекта

```
.
├── main.py              # Точка входа, основной игровой цикл
├── editor.py            # Редактор уровней
├── map.json             # Файл уровня (создаётся редактором)
└── scripts/
    ├── utils.py         # Вспомогательные функции загрузки изображений, класс Animation
    ├── tilemap.py       # Тайловая карта: рендер, физика, сохранение/загрузка JSON, автотайлинг
    ├── entities.py      # Базовая физическая сущность PhysicsEntity и класс Player
    ├── clouds.py        # Классы Cloud и Clouds — параллакс-фон из облаков
    └── particles.py     # Класс Particle — система частиц (листья, эффект рывка)
```

### Описание модулей

- **`main.py`** — инициализация Pygame, загрузка ресурсов (изображения, анимации, звуки), создание облаков, тайловой карты и игрока. Основной игровой цикл: обработка ввода, обновление камеры, рендер всех объектов.
- **`tilemap.py`** — хранит тайловую карту в словаре. Реализует рендер, получение тайлов вокруг позиции, физические коллизии, сохранение и загрузку уровня в JSON, алгоритм автотайлинга.
- **`utils.py`** — функции `load_image` и `load_images` для загрузки ресурсов, класс `Animation` для покадровой анимации с поддержкой цикла.
- **`entities.py`** — базовый класс `PhysicsEntity` с движением и обнаружением коллизий; класс `Player`, наследующий его и добавляющий прыжок от стены, рывок и спавн частиц.
- **`clouds.py`** — классы `Cloud` и `Clouds` для отрисовки параллакс-облаков на фоне.
- **`particles.py`** — класс `Particle` для отрисовки анимированных частиц (листья, эффект рывка персонажа).

## Сборка исполняемого файла

Для сборки `.exe` используется PyInstaller:

```bash
pip install pyinstaller
pyinstaller --onefile main.py      # сборка игры
pyinstaller --onefile editor.py    # сборка редактора
```

Готовые файлы появятся в директории `dist/`.
