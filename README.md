# 2D Platformer на Pygame

## Требования к окружению

•	Python версии 3.10 и выше;
•	установленная библиотека Pygame (pip install pygame);

## Запуск игры

1.	Убедиться, что файл уровня level01.json находится в той же директории, что и main.py.
2.	Запустить игру командой:
python main.py

## Управление в игре

•	A или стрелка влево — движение персонажа влево;
•	D или стрелка вправо — движение персонажа вправо;
•	W, стрелка вверх или SPACE — прыжок;
•	Left Shift или X — рывок в сторону текущего движения;
•	Esc — выход из игры.
Цель игрока — добежать до финишного блока уровня. При достижении финиша отображается окно «Level passed!» с предложением выбрать дальнейшее действие.

## Запуск редактора уровней

Редактор уровней запускается отдельной программой:
python editor.py

В редакторе пользователь может создавать и изменять карту, размещая на сетке тайлы разных типов (обычные платформы и финишный блок), а затем сохранять её в файл level01.json.

## Управление в редакторе

•	W, A, S, D — перемещение камеры по уровню;
•	левая кнопка мыши — установить тайл текущего типа;
•	правая кнопка мыши — удалить тайл в выбранной клетке;
•	клавиша S — сохранить карту в файл JSON;
•	клавиша L — загрузить карту из файла JSON;
•	клавиша G — показ или скрытие сетки;
•	клавиша Tab — переключение типа редактируемого тайла (обычный/финиш);
•	Esc — выход из редактора.
